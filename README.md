Customer Churn Prediction

# Project Overview
This project focuses on predicting customer churn using Machine Learning techniques.  
Customer churn refers to customers who stop using a company's service. By predicting churn, businesses can take proactive steps to retain customers.

---

# Objective
- Analyze customer data
- Identify key factors affecting churn
- Build a machine learning model to predict churn

---

# Dataset
- Dataset: Telco Customer Churn Dataset
- Features include:
  - Customer demographics (gender, SeniorCitizen)
  - Account information (tenure, contract)
  - Services (InternetService, OnlineSecurity, etc.)
  - Billing (MonthlyCharges, TotalCharges)
